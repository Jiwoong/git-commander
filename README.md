# git commander

A git tool with an easy terminal interface.

![ScreenShot](https://raw.githubusercontent.com/golbin/git-commander/master/doc/git-commander.gif)

# Features

- [x] git status
- [x] git add [files]
- [x] git reset -- [files]
- [x] git commit [files]
- [x] git log
- [x] git reset <commit>
- [x] git diff file
- [ ] git branch
- [ ] git push

# Install & Usage

```bash
$ npm install -g git-commander
$ git-commander
```
