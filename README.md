# git commander

A git tool with a easy terminal interface.

![ScreenShot](https://raw.githubusercontent.com/golbin/git-commander/master/doc/git-commander.gif)

# Features

- [x] git status
- [x] git add [files]
- [x] git rest -- [files]
- [x] git commit [files]
- [ ] git log
- [ ] git reset <commit>
- [ ] git diff [<file>]

# Install & Usage

```bash
$ npm install -g git-commander
$ git-commander
```
